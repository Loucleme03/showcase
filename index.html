Demo 3 - ArcGIS Maps SDK for Javascript

This demonstration uses arcGIS Maps SDK for Javascript where it displays 2 raster images of the exact same clipping extent to show the difference in the landscape between 1984 and 2021. The raster images were sourced from USGS. To see the difference you need to turn off visibility for one of the raster images.

Demo 3

Demo 4 - esri-Leaflet Map

This demonstration uses the same raster images to display the changes in the landscape of the same clipping extent as used in demo 4. It uses esri-Leaflet instead and has a slider function to display the difference in the landscapes overtime.
