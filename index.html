Lou's Portfolio

Thank you for visiting. The demos in this showcase were all created during my education at Fleming College in 2023 from the GIS-applications specialist program. I created this webpage to illustrate a brief snapshot of some of the things I learnt from the program.

It hightlights some work in autoCAD map 3D, ArcGIS PRO, web design, map api's and JavaScript. I could not include any arcGIS pro or online demos other than a storymap pdf (Demo 5) as it was a school account and I no longer have access to the account. To see how the Javascript for the following demos were cretead, please click here and scroll to the demo you want to look at.

Below walks you through the demo's and you can click the button to go to the example or use the navbar.

Demo 1 - Javascript API

In this demonstration I created a javascript api googlemap. It is set to display a road trip in england that I went on with my family.

Demo 2 - CAD Drawings

This demonstration shows some of the CAD drawings that I made with AutoCAD map 3D 2023. It is displayed as a slideshow.

Demo 3 - Esri-Leaflet Map

This demonstration uses Esri Leaflet Maps where it displays 2 raster images of the exact same clipping extent to show the difference in the landscape between 1984 and 2021. The raster images were sourced from USGS. To see the difference you can slide or click on the slider to show changes in landscape overtime. The image changes opacity from as you change the year of the slider to help compare.

Demo 3

Demo 4 - Remote Sensing

This will display a series of rendered images using PCI Geomatics Catalyst and some of the various techniques / processes from assignments that I completed in the program. NOTE: the quality of the images were reduced as they are snips of the original images and converted to jpg.

This will open a pdf version of a storymap that was created by building the shapefiles in ArcGIS PRO and then put together using ArcGIS online - Storymaps app. Some of the pictures are cut off but it still shows some of the tools that were explored in my spatial analysis course and ability to use arcGIS online apps.
